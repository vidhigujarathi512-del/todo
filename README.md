# 🌿 BloomBoard - Notion Inspired To-Do App

BloomBoard is a clean and minimalist task management application inspired by Notion's to-do lists. It helps users organize tasks, track progress, and manage productivity with a simple and intuitive interface.

## 🚀 Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed using checkboxes
* Search tasks instantly
* Filter tasks by:

  * All
  * Active
  * Completed
* Progress tracking with a dynamic progress bar
* Data persistence using Local Storage
* Responsive design for desktop and mobile devices

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## ⚙️ How It Works

1. Users create tasks using the input form.
2. Tasks are stored in an array and automatically saved to Local Storage.
3. The application dynamically renders tasks on the page.
4. Users can:

   * Complete tasks using checkboxes
   * Edit task titles
   * Delete unwanted tasks
5. Progress is calculated based on completed tasks.
6. All data remains available even after refreshing the browser.
